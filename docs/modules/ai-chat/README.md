# AI Travel Concierge Chat

> **Feature ID:** F01  
> **Status:** Planned  
> **Scope:** MVP

## Overview

Full-screen WebSocket-driven AI chat interface powered by a Python LangGraph + Claude agent. Renders text, trip cards, hotel cards, action buttons, and payment QR codes inline. This is the core "Vibe Booking" experience.
