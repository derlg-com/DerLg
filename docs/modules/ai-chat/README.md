# AI Travel Concierge Chat

> **Feature ID:** F10–F16  
> **Status:** Planned  
> **Scope:** MVP

## Overview

DerLg's core differentiator: a conversational AI concierge powered by LangGraph + Claude that lets travelers discover, plan, and book trips through natural language chat. Supports text, trip cards, hotel cards, action buttons, and payment QR codes inline.
